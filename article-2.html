Basics of Networking

Building a solid foundation by understanding the core concepts that power the internet and all connected devices.

By Zakaria Rami July 23, 2025

What is a Network?

At its core, a computer network is a group of two or more interconnected computing devices that can exchange data and share resources. These devices can be anything from servers, desktops, and laptops to smartphones and IoT devices. The connections between them can be wired (using Ethernet cables) or wireless (using Wi-Fi). The internet itself is the largest example of a network—a global network of networks.

The OSI Model: A Conceptual Framework

To understand how networks function, it's helpful to know about the OSI (Open Systems Interconnection) model. It's a conceptual framework that standardizes the functions of a telecommunication or computing system into seven abstract layers. Each layer serves a specific purpose and communicates with the layers directly above and below it.

  • Layer 7 (Application): Where network applications like web browsers and email clients operate.
  • Layer 4 (Transport): Provides reliable data transfer with protocols like TCP and UDP.
  • Layer 3 (Network): Handles logical addressing and routing (e.g., IP addresses).
  • Layer 2 (Data Link): Manages physical addressing (e.g., MAC addresses).
  • Layer 1 (Physical): The actual hardware—cables, switches, and wireless signals.

Key Components of a Network

IP Addresses & Subnetting

An IP (Internet Protocol) address is a unique numerical label assigned to each device on a network. It's like a street address for your computer. For example, 192.168.1.10 is a common private IP address. Networks are often divided into smaller sub-networks, or "subnets," to improve organization and performance.

Routers and Switches

These are the traffic cops of a network.

A switch operates at Layer 2 and connects devices within the same local network (LAN). It uses MAC addresses to forward data to the correct device.
A router operates at Layer 3 and connects different networks together. It uses IP addresses to route traffic between networks, including from your local network to the internet.

DNS: The Internet's Phonebook

When you type www.google.com into your browser, your computer doesn't magically know where to find it. It uses the Domain Name System (DNS) to look up the IP address associated with that domain name. DNS servers translate human-readable names into machine-readable IP addresses, making the internet navigable.

Conclusion

This is just a brief overview, but understanding these fundamental concepts—the OSI model, IP addressing, and the roles of key hardware—is the first step to mastering the world of networking. Each of these topics can be explored in much greater depth, forming the foundation for more advanced subjects like network security, cloud computing, and wireless technologies.
